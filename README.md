# Twin Flames Masters - Sitio Web

Sitio web profesional para coaching de Ascensión de Llamas Gemelas, construido con React y Vite.

## Características

- ✨ Diseño moderno y responsivo
- 🎨 Interfaz limpia con Tailwind CSS
- ⚡ Construido con Vite para máxima velocidad
- 📱 Optimizado para móvil
- 🚀 Listo para desplegar en Vercel

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

El sitio estará disponible en `http://localhost:5173`

## Build para Producción

```bash
npm run build
```

Los archivos compilados estarán en la carpeta `dist/`

## Despliegue a Vercel

### Opción 1: Desde la CLI
```bash
npm i -g vercel
vercel
```

### Opción 2: GitHub Integration
1. Sube este repositorio a GitHub
2. Conecta tu repositorio en Vercel.com
3. Vercel detectará automáticamente que es un proyecto Vite y lo configurará

## Personalización

### Editar Contenido

Todos los componentes están en `src/components/`. Puedes editar el contenido directamente en los archivos JSX.

### Cambiar Colores

Edita `tailwind.config.js` para personalizar los colores y temas.

## Tecnologías Utilizadas

- React 19
- Vite 8
- Tailwind CSS 4
